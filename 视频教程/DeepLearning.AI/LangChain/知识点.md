
# Memory
配置memory，
> conversation = ConversationChain(
    llm=llm, 
    memory = memory,
    verbose=True
)


## ConversationBufferMemory
> memory = ConversationBufferMemory()

将聊天记录全部放在内部中。

## ConversationBufferWindowMemory
> memory = ConversationBufferWindowMemory(k=1)               
与ConversationBufferMemory增加了参数，用来保存指定轮次的对话数。
